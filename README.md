Rever
=====

Rever is a work-in-progress reversible programming language. Examples of such languages include [Janus](https://en.wikipedia.org/wiki/Janus_(time-reversible_computing_programming_language)) and [rFun](http://topps.diku.dk/pirc/?id=rfun).
